Sociyall - an umbrella social app for aggregating multiple raw social media feeds

Posted by Eprobito on Jan 12, 2023 TCP. 4 comments

Sociyall was built with the understanding that users have a need to interact with specific social media feeds at specific points in time. So, the app provides a mechanism to “follow” based on

People: Specific people that you would like to keep updated about

Events: All social media around an event is tracked. After the event is over, this is turned off

Location: All social media related to a location is allowed in the app’s feed

Things: Specific “product” related social media is tracked.

At any given point in time, the user can change, turn on, turn on any of the PELT feeds he/she wants to continue receiving.

Currently, the social media enabled on Sociyall include Facebook, Twitter and Instagram.
